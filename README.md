# Trend Follower

A Python-based trend-following trading application for analyzing market data and implementing systematic trading strategies.

## Setup

1. Create and activate virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Launch Jupyter:
   ```bash
   jupyter lab
   ```

## Project Structure

- `notebooks/` - Jupyter notebooks for analysis and experimentation
- `src/` - Python source code
- `tests/` - Test files
- `data/` - Data files
